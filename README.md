# chsoares' Kali setup

1. Pimp My Kali
```
cd /opt
rm -rf pimpmykali/
git clone https://github.com/Dewalt-arch/pimpmykali
cd pimpmykali
sudo ./pimpmykali.sh
```

2. Setup.sh
```
cd /root; mkdir lab; cd lab
git clone https://github.com/chsoares/kali_setup.git
./setup.sh
```

3. Cleanup
```
cleanup
```
